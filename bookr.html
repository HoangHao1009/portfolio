BOOK RECOMMEDATION SYSTEM PROJECT

Business Understanding

This project can helps:

+ Recommend a book based on users choice.

+ Represent the book market with vivid charts.

So it's useful for both book seekers or book dealer by providing useful information of book market.

Analytics Approach

+ Recommend a book by similarity of user's choices and book data.

+ Calculate useful metrics for book market understanding and visualize it.

Data preparation

+ This is an application of my another project: Tiki Web Crawler (https://github.com/HoangHao1009/hcrawler).

Modelling

+ Unsupervised learning: KMeans Clustering for segment book.

+ Deep Learning (NLP): Vectorize book's describes.
